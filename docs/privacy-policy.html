🔒 Politique de Confidentialité

Application Scandio - Dernière mise à jour : Janvier 2026

En résumé : Scandio respecte votre vie privée. Vos données de cartes cadeaux restent sur votre appareil et ne sont jamais collectées par nos serveurs.

1. Introduction

Bienvenue sur Scandio, votre application de gestion de cartes cadeaux. Cette politique de confidentialité explique comment nous traitons vos informations lorsque vous utilisez notre application mobile.

Scandio est développée et maintenue par Damien Lebas ("nous", "notre", "nos"). En utilisant Scandio, vous acceptez les pratiques décrites dans cette politique.

2. Données collectées

Données stockées localement sur votre appareil :

  • Informations des cartes cadeaux (nom de l'enseigne, code-barres, solde, notes)
  • Historique des transactions que vous saisissez
  • Préférences de l'application

Données NON collectées par nos serveurs :

  • Aucune donnée personnelle
  • Aucune information de carte cadeau
  • Aucune donnée de localisation
  • Aucun identifiant publicitaire
  • Aucune donnée d'utilisation
💡 Important : Toutes vos données de cartes cadeaux sont stockées exclusivement sur votre appareil. Nous n'avons aucun accès à ces informations.

3. Fonctionnalités optionnelles

Synchronisation Cloud (optionnelle) :

Si vous choisissez d'activer la synchronisation cloud, vos données seront stockées de manière sécurisée sur nos serveurs hébergés par Supabase. Ces données sont :

  • Chiffrées en transit (HTTPS/TLS)
  • Stockées dans des centres de données sécurisés (AWS)
  • Accessibles uniquement avec votre compte
  • Supprimables à tout moment via l'application

Données synchronisées : informations des cartes cadeaux (nom, code-barres, solde, notes), historique des transactions.

Partage de cartes (optionnel) :

La fonction de partage vous permet d'envoyer une carte cadeau à un proche. Les données partagées transitent de manière sécurisée et sont automatiquement supprimées après utilisation.

4. Utilisation des données

Les données stockées localement sont utilisées uniquement pour :

  • Afficher vos cartes cadeaux et leurs soldes
  • Générer les codes-barres pour utilisation en magasin
  • Synchroniser entre vos appareils (si activé)

5. Sécurité

Nous prenons la sécurité de vos données au sérieux :

  • Les données restent sur votre appareil par défaut
  • Les communications réseau utilisent le chiffrement HTTPS
  • L'application fonctionne hors ligne
  • Aucune création de compte requise

6. Droits des utilisateurs

Vous avez le droit de :

  • Accéder à vos données : toutes vos données sont visibles dans l'application
  • Modifier vos données : vous pouvez éditer toutes les informations
  • Supprimer vos données : désinstallez l'application ou supprimez les cartes individuellement
  • Exporter vos données : disponible via les paramètres de l'application

7. Services tiers

Scandio utilise les services suivants :

  • Supabase (optionnel) : pour la synchronisation cloud et l'authentification
    → Politique de confidentialité Supabase

Ces services sont soumis à leurs propres politiques de confidentialité. Nous vous encourageons à les consulter via les liens ci-dessus.

8. Hébergement des données

Lorsque la synchronisation cloud est activée, vos données sont hébergées par Supabase sur l'infrastructure Amazon Web Services (AWS), située dans l'Union Européenne (région eu-west). Cela garantit la conformité avec le RGPD.

9. Enfants

Scandio n'est pas destinée aux enfants de moins de 13 ans. Nous ne collectons pas sciemment d'informations auprès d'enfants.

10. Modifications

Nous pouvons mettre à jour cette politique de confidentialité occasionnellement. Les modifications seront publiées sur cette page avec une date de mise à jour.
